# GIST AgentCrew — Onboarding Guide

새로 합류하신 것을 환영합니다!

## 시작하기 전에

1. [MANIFESTO.md](MANIFESTO.md)를 읽고 팀의 철학을 이해해주세요.
2. [ROADMAP.md](ROADMAP.md)를 읽고 현재 진행 상황을 파악해주세요.

## 환경 세팅

### 1. GitHub

- GitHub 계정이 없다면 생성
- 팀장에게 GitHub ID 전달 → Organization 초대
- 초대 수락 후 repo 접근 확인

### 2. Discord

- 팀 Discord 서버 참여 (초대 링크는 팀장에게 문의)
- 자기소개 채널에 간단한 소개 작성

### 3. 개발 환경

- Python 3.10 이상 설치
- Git 설치 및 기본 사용법 숙지
- 로컬에서 오픈소스 LLM 실행 환경 구축 (ollama 등)

## 인프라 구조

```
┌──────────────────────┐
│  DGX Spark 2대       │  ← 큰 모델 서빙, 팀 개발용
│  (분산 컴퓨팅, vLLM)   │
└──────────┬───────────┘
           │ 내부 네트워크
┌──────────▼───────────┐
│  Mobile X (엣지)      │  ← 교육 실습용
│  RTX 3000, 소형 모델  │
└──────────────────────┘
```

- 큰 모델은 DGX Spark에서 서빙
- 작은 모델(7B~8B 양자화)은 Mobile X에서 로컬 실행 가능
- 데이터는 내부 네트워크 안에서만 이동 (외부 API 사용 금지)

## 핵심 원칙

- **개발 과정** — API, AI 도구 자유롭게 사용 OK (효율적으로 개발하세요)
- **최종 산출물** — 외부 API 의존 없이 로컬/자체 서버에서 동작해야 합니다
- **Open Source** — 산출물은 공개합니다
