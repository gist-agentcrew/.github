# GIST AgentCrew — Governance

## 역할 구조

| 역할 | 책임 |
|------|------|
| **교수** | 연구 방향 감수, 최종 의사결정 |
| **멘토** | 기술 지도, 팀원 코드 리뷰 |
| **팀장** | 팀 운영, 일정 관리, 인프라 관리 |
| **팀원** | Agent 개발, 교육 자료 제작 |

## 의사결정

### 일상적 결정
- 각자 판단하여 진행
- 예: 본인 Agent의 기술 선택, 코드 구조

### 팀 차원의 결정
- Discord 또는 회의에서 논의
- 팀원 과반수 동의로 결정
- 예: 공용 인프라 변경, 커리큘럼 구조

### 중요한 결정
- 교수, 멘토, 팀장 협의
- 예: 팀 방향 전환, 외부 발표, 대외 협력

## 회의

- **정기 회의:** 주 1회
- **회의록:** GitHub Discussions 또는 Discord에 기록
- 불참 시 사전 공유

## GitHub 권한

| 역할 | GitHub 권한 |
|------|------------|
| 교수 | Owner |
| 멘토 | Maintainer |
| 팀장 | Owner |
| 팀원 | Write |

## 갈등 해결

1. 당사자 간 대화
2. 해결 안 되면 팀장 중재
3. 그래도 안 되면 멘토/교수 판단
