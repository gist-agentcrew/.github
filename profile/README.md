# GEAR

> *"Build your own Agent. Own your data."*

**GIST Educational Agentic Researchers** — 광주과학기술원(GIST) 소속 AI Agent 연구·교육 팀입니다. (Sub-group of `Dream-AI`)

## What We Do

- 오픈소스 LLM 기반, 로컬에서 동작하는 AI Agent를 연구합니다.
- 고등학생과 일반인을 대상으로 AI Agent 교육을 진행합니다.
- 모든 산출물을 오픈소스로 공개합니다.

## Our Principles

| Principle | Description |
|-----------|-------------|
| **No External API** | 외부 API에 데이터를 보내지 않습니다 |
| **Local First** | 로컬 실행을 기본으로 합니다 |
| **Open Source** | 모든 산출물을 공개합니다 |
| **Anyone Can Build** | 누구나 만들 수 있도록 설계합니다 |
